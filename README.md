# ⌨️ Estudo de Captura de Eventos de Teclado em Python

Este projeto foi desenvolvido com fins acadêmicos para estudo de captura de eventos de entrada (teclado), como parte da disciplina de Segurança da Informação e Sistemas Operacionais.

---

## 📌 Objetivo

O objetivo principal deste projeto é compreender:

* Como sistemas operacionais tratam eventos de entrada (I/O)
* Interceptação de eventos de teclado em baixo nível
* Estrutura de programas em linguagem C voltados para monitoramento
* Implicações de segurança relacionadas à captura de dados

---

## 🛠️ Tecnologias Utilizadas

* Linguagem: Python
* Conceitos:

  * Manipulação de entrada/saída (I/O)
  * Programação de baixo nível
  * Interação com o sistema operacional

---

## ⚙️ Funcionamento

O programa realiza a captura de eventos de teclado e registra as entradas de forma contínua, permitindo analisar o fluxo de dados gerado pelo usuário.

---
